# Music Recommender System

A simple music recommendation system built using audio features such as danceability, energy, tempo, etc.
Dataset contains Spotify songs with language tags.

## Features
✔ Content-based similarity using KNN  
✔ Language filtering (EN, HI, TE etc)  
✔ Command-line interface  

## Tech Stack
- Python
- Pandas
- Scikit-learn
